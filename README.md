## 📌 About the Project

**Data Analytics Questions** is an interactive **Streamlit** application designed to help students, beginners, and aspiring Data Analysts practice real-world **Pandas**, **NumPy**, and **Data Analysis** concepts.

Instead of reading theory, you solve practical questions using actual datasets such as:

- 🚗 Car Dataset
- 🏏 IPL Dataset
- 🎵 Spotify Dataset
- 🏠 House Price Dataset
- 🛒 Supermarket Sales Dataset
- 📏 Heights & Weights Dataset
- 🎬 Movies Dataset

Every question includes:
- ✅ Solution
- ✅ Output
- ✅ Optional Code View
- ✅ Interactive Dashboard

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/data-analytics-questions.git
```

Go to project folder

```bash
cd data-analytics-questions
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run Streamlit

```bash
streamlit run questions.py
```
